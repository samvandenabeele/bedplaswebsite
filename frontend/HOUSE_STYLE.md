# House style

- Use Tailwind utility classes first; avoid custom CSS unless the layout really needs it.
- Keep the app dark, calm, and high contrast.
- Center the active form card both vertically and horizontally.
- Prefer rounded-2xl or larger cards with soft borders and subtle blur.
- Keep labels short and controls easy to reach for repetitive data entry.
- Use a simple two-state switcher for admin/user views instead of deep navigation.
- Favor compact layouts, sensible defaults, and clear primary actions.
- Make the main action button obvious; secondary actions should stay visually lighter.
